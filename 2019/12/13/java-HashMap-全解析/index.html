HashMap的内部数据结构

  1. 默认初始大小16

  2. 最大容量2^30次方

  3. 默认负载因子0.75

  4. 树化临界值8
    执行put操作的时候，会出现桶碰撞的情况，这时候桶索引值相同的键值对会以一个链表的形式存在于hash桶中，但是当链表长度很长的时候，
    查找的性能会很低.当链表的长度超过TREEIFY_THRESHOLD（8）的时候，链表会树化，即通过treeifyBin()方法转换成红黑树。
